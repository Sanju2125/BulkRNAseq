# BulkRNAseq
RNA-seq analysis of breast cancer samples

# RNA-seq Analysis Pipeline

## Overview
This repository contains scripts and metadata for RNA-seq analysis using:

- HISAT2 (alignment)
- StringTie (reference-guided transcriptome assembly)
- featureCounts (gene-level quantification)
- DESeq2 (differential expression)
